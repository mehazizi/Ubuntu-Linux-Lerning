# فصل ۱۷ - آرشیو و فشرده‌سازی | خلاصه

## ابزارها
| ابزار | پسوند | توضیح |
|-------|-------|-------|
| `tar` | .tar | آرشیو (بدون فشرده‌سازی) |
| `gzip` | .gz | فشرده‌سازی سریع |
| `bzip2` | .bz2 | فشرده‌سازی بهتر، کندتر |
| `xz` | .xz | بهترین فشرده‌سازی، کندترین |
| `zip` | .zip | سازگار با ویندوز |

## ترکیب tar با فشرده‌سازی
| پسوند | دستور |
|-------|-------|
| `.tar.gz` یا `.tgz` | tar + gzip |
| `.tar.bz2` | tar + bzip2 |
| `.tar.xz` | tar + xz |

## گزینه‌های مهم tar
| گزینه | معنی |
|-------|------|
| `c` | ساخت آرشیو |
| `x` | استخراج |
| `t` | لیست محتوا |
| `f` | نام فایل |
| `v` | نمایش جزئیات |
| `z` | gzip |
| `j` | bzip2 |
| `J` | xz |
| `p` | حفظ مجوزها |
