# فصل ۱ - مبانی لینوکس | آزمایشگاه

## تمرین ۱: شناخت سیستم
```bash
# اطلاعات سیستم‌عامل را بخوانید
uname -a
cat /etc/os-release
lsb_release -a

# نسخه هسته را یادداشت کنید:
# ________________________________
```

## تمرین ۲: سیستم راهنما
```bash
# راهنمای دستور ls را باز کنید
man ls
# با کلیدهای j/k یا فلش بالا/پایین حرکت کنید
# با q خارج شوید

# دستوری برای نمایش تقویم پیدا کنید
apropos calendar

# توضیح یک‌خطی دستور cp را ببینید
whatis cp
```

## تمرین ۳: متغیرهای محیطی
```bash
# همه متغیرها را ببینید
env | head -20

# متغیر خودتان بسازید
export MY_NAME="کاربر من"
echo $MY_NAME

# بررسی کنید آیا PATH شامل /usr/local/bin هست؟
echo $PATH | tr ':' '
'
```

## تمرین ۴: تاریخچه و Exit Status
```bash
# یک دستور موفق اجرا کنید
ls /tmp
echo $?          # باید 0 باشد

# یک دستور ناموفق اجرا کنید
ls /nonexistent
echo $?          # باید غیر صفر باشد

# تاریخچه را ببینید
history | tail -10
```

## تمرین ۵: میانبرهای کیبورد
```
Ctrl+R را فشار دهید و "ls" را تایپ کنید → جستجو در تاریخچه
Tab را بعد از تایپ "cat /etc/os" فشار دهید → تکمیل خودکار
Ctrl+C را برای لغو دستور زیر فشار دهید:
  sleep 100
```
