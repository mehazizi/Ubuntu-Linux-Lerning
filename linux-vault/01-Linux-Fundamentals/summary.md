# فصل ۱ - مبانی لینوکس | خلاصه

## لینوکس چیست؟
لینوکس یک سیستم‌عامل متن‌باز و رایگان است که در ۱۹۹۱ توسط **لینوس توروالدز** ساخته شد.
هسته (Kernel) لینوکس قلب سیستم است و بین نرم‌افزار و سخت‌افزار ارتباط برقرار می‌کند.

## GNU چیست؟
پروژه‌ای که ریچارد استالمن در ۱۹۸۳ شروع کرد تا یک سیستم‌عامل کاملاً آزاد بسازد.
ابزارهایی مثل gcc، bash، glibc همه از پروژه GNU هستند.

## GNU/Linux
ترکیب **هسته لینوکس** + **ابزارهای GNU** = سیستم‌عامل کامل
به همین دلیل برخی آن را GNU/Linux می‌نامند نه فقط Linux.

## معماری لینوکس
```
┌─────────────────────────┐
│      User Space         │  ← برنامه‌ها، Shell، ابزارها
├─────────────────────────┤
│      System Call        │  ← رابط بین User و Kernel
├─────────────────────────┤
│      Kernel Space       │  ← مدیریت حافظه، پردازنده، دستگاه‌ها
├─────────────────────────┤
│      Hardware Layer     │  ← CPU، RAM، دیسک، شبکه
└─────────────────────────┘
```

## توزیع (Distribution)
هسته لینوکس + ابزارها + مدیر بسته + رابط کاربری = توزیع
نمونه‌ها: Ubuntu، Debian، Fedora، Arch، CentOS

## Shell و Terminal
- **Shell**: مفسر دستورات (bash، zsh، fish)
- **Terminal**: پنجره‌ای که Shell را نشان می‌دهد
- **Console**: ترمینال فیزیکی یا مجازی سیستم
- **TTY**: دستگاه ترمینال (tty1 تا tty6)
- **Prompt**: نشانه آمادگی Shell برای دریافت دستور

## ساختار دستور لینوکس
```
command  [options]  [arguments]
   ls      -la       /var/log
```

## Exit Status
- **0** = موفق
- **هر عدد غیر صفر** = خطا
- بررسی: `echo $?`

## متغیرهای محیطی مهم
| متغیر | کاربرد |
|-------|---------|
| `$PATH` | مسیر جستجوی دستورات |
| `$HOME` | پوشه خانگی کاربر |
| `$USER` | نام کاربر جاری |
| `$SHELL` | Shell پیش‌فرض |
| `$PWD` | پوشه جاری |
