# فصل ۱۸ - SSH | خلاصه

## SSH چیست؟
Secure Shell - پروتکل رمزنگاری‌شده برای دسترسی امن از راه دور.

## انواع احراز هویت
| روش | امنیت | سهولت |
|-----|-------|-------|
| رمز عبور | متوسط | ساده |
| کلید عمومی | بالا | بعد از تنظیم آسان |
| Certificate | بسیار بالا | پیچیده |

## انواع کلید SSH
| نوع | توضیح |
|-----|-------|
| RSA | قدیمی، حداقل ۴۰۹۶ بیت |
| ECDSA | بیضوی، سریع‌تر |
| Ed25519 | مدرن، توصیه‌شده |

## فایل‌های مهم
| فایل | کاربرد |
|------|---------|
| `~/.ssh/id_ed25519` | کلید خصوصی |
| `~/.ssh/id_ed25519.pub` | کلید عمومی |
| `~/.ssh/authorized_keys` | کلیدهای مجاز برای ورود |
| `~/.ssh/config` | تنظیمات اتصال |
| `/etc/ssh/sshd_config` | پیکربندی سرور SSH |
