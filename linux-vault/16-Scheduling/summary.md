# فصل ۱۶ - زمان‌بندی وظایف | خلاصه

## ابزارهای زمان‌بندی
| ابزار | کاربرد |
|-------|---------|
| `cron` | وظایف تکراری (هر دقیقه تا هر سال) |
| `systemd timer` | جایگزین مدرن cron |
| `at` | اجرای یک‌بار در زمان مشخص |

## فرمت crontab
```
دقیقه  ساعت  روزِماه  ماه  روزِهفته  دستور
  *      *      *       *      *      command

# مثال‌ها:
0 2 * * *        → هر روز ساعت ۰۲:۰۰
*/5 * * * *      → هر ۵ دقیقه
0 0 * * 0        → هر یکشنبه ساعت ۰۰:۰۰
0 9 1 * *        → اول هر ماه ساعت ۰۹:۰۰
30 8 * * 1-5     → شنبه تا چهارشنبه ساعت ۰۸:۳۰
```

## پوشه‌های cron سیستمی
| پوشه | زمان اجرا |
|------|-----------|
| `/etc/cron.hourly/` | هر ساعت |
| `/etc/cron.daily/` | هر روز |
| `/etc/cron.weekly/` | هر هفته |
| `/etc/cron.monthly/` | هر ماه |
