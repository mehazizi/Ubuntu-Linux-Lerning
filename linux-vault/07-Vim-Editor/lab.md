# فصل ۷ - ویرایشگر Vim | آزمایشگاه

## تمرین ۱: آموزش داخلی Vim
```bash
vimtutor
# این آموزش تعاملی تمام مبانی را پوشش می‌دهد
# حدود ۳۰ دقیقه وقت بگذارید
```

## تمرین ۲: ویرایش فایل پیکربندی
```bash
# یک فایل تمرینی بسازید
cp /etc/ssh/sshd_config ~/sshd_config_practice

vim ~/sshd_config_practice
# تمرین کنید:
# 1. با /Port خط Port را پیدا کنید
# 2. با n بین نتایج جابجا شوید
# 3. یک خط را با dd حذف کنید
# 4. با u برگردانید
# 5. با :q! بدون ذخیره خارج شوید
```

## تمرین ۳: جستجو و جایگزینی
```bash
vim ~/sshd_config_practice
# جایگزین کنید:
# :%s/#Port/Port/g
# با :set number شماره خط را فعال کنید
# با :wq ذخیره و خروج کنید
```

## تمرین ۴: ~/.vimrc شخصی
```bash
vim ~/.vimrc
# این تنظیمات را اضافه کنید:
# set number
# set tabstop=4
# syntax on
# :wq
```
