# فصل ۹ - پیکربندی شبکه | خلاصه

## مفاهیم پایه
- **Interface**: رابط شبکه (eth0، ens3، enp3s0)
- **IP Address**: آدرس منحصربه‌فرد در شبکه
- **Subnet Mask**: محدوده شبکه (مثلاً /24 = 255.255.255.0)
- **Gateway**: دروازه خروج از شبکه محلی
- **DNS**: تبدیل نام به آدرس IP

## ابزارهای پیکربندی در اوبونتو
| ابزار | کاربرد |
|-------|---------|
| `Netplan` | پیکربندی اصلی شبکه (فایل YAML) |
| `NetworkManager` | مدیریت پویا (در دسکتاپ) |
| `iproute2` | دستورات ip |

## فایل‌های مهم
| فایل | کاربرد |
|------|---------|
| `/etc/netplan/*.yaml` | پیکربندی شبکه |
| `/etc/hosts` | نگاشت نام محلی |
| `/etc/resolv.conf` | تنظیمات DNS |
| `/etc/hostname` | نام هاست |

## مفاهیم پیشرفته
- **VLAN**: شبکه مجازی روی یک رابط فیزیکی
- **Bridge**: اتصال چند رابط در سطح لایه ۲
- **Bond**: ترکیب چند رابط برای پهنای باند بیشتر یا Failover
- **MTU**: حداکثر اندازه بسته (پیش‌فرض ۱۵۰۰)
