# فصل ۷ - ویرایشگر Vim | عیب‌یابی

## مشکل: گیر کردن در Vim
```
اگر نمی‌دانید در کدام حالت هستید:
1. چند بار Esc بزنید تا به Normal Mode برگردید
2. برای خروج اضطراری: Esc → :q!
```

## مشکل: فایل swap وجود دارد
```
E325: ATTENTION - Found a swap file
```
```bash
# فایل swap را حذف کنید
rm ~/.filename.swp
# یا در پیام، O (open anyway) یا D (delete swap) انتخاب کنید
```

## مشکل: تغییرات ذخیره نمی‌شود
```bash
# بررسی مجوز فایل
ls -la file.txt
# ذخیره با sudo
:w !sudo tee %
```

## مشکل: رنگ‌بندی کار نمی‌کند
```vim
:syntax on
:set syntax=python    " تنظیم دستی نوع فایل
```

## نکته: خروج سریع در هر وضعیت
```
Esc Esc :q!   ← خروج اضطراری بدون ذخیره
```
