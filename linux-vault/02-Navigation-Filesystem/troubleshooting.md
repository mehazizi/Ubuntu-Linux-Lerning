# فصل ۲ - ناوبری و فایل‌سیستم | عیب‌یابی

## مشکل: cd به پوشه نمی‌رود
```bash
# بررسی وجود پوشه
ls -la /path/to/folder

# بررسی مجوز
stat /path/to/folder

# مطمئن شوید مسیر را درست تایپ کرده‌اید
pwd && ls
```

## مشکل: فایل مخفی پیدا نمی‌شود
```bash
# حتماً از -a استفاده کنید
ls -la ~
ls -la /etc | grep "^\."
```

## مشکل: Wildcard کار نمی‌کند
```bash
# اگر نتیجه نداد بررسی کنید
echo *.txt             # ببینید چه نامه‌ای match می‌شود
ls -la | grep ".txt"   # جستجوی مستقیم
```
