# فصل ۳ - فایل‌ها و پوشه‌ها | خلاصه

## انواع فایل در لینوکس
| نوع | نماد در ls | توضیح |
|-----|-----------|-------|
| فایل عادی | `-` | متن، باینری |
| پوشه | `d` | دایرکتوری |
| لینک نمادین | `l` | Symbolic Link |
| دستگاه بلاک | `b` | دیسک |
| دستگاه کاراکتر | `c` | ترمینال |
| سوکت | `s` | ارتباط بین فرآیندها |
| FIFO | `p` | Pipe نام‌دار |

## Hard Link در مقابل Soft Link
| ویژگی | Hard Link | Soft Link |
|-------|-----------|-----------|
| به فایل اصلی وابسته؟ | خیر | بله |
| روی پارتیشن دیگر؟ | خیر | بله |
| به پوشه؟ | خیر | بله |
| بعد از حذف فایل اصلی؟ | کار می‌کند | شکسته می‌شود |

## دستور find - گزینه‌های مهم
| گزینه | کاربرد |
|-------|---------|
| `-name` | جستجو بر اساس نام |
| `-type f/d/l` | نوع فایل |
| `-size +100M` | بزرگتر از ۱۰۰ مگابایت |
| `-mtime -7` | تغییر در ۷ روز اخیر |
| `-user username` | متعلق به کاربر |
| `-perm 644` | مجوز خاص |
