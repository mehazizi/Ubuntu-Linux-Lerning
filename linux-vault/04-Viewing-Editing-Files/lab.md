# فصل ۴ - مشاهده و ویرایش فایل‌ها | آزمایشگاه

## تمرین ۱: کار با less
```bash
less /var/log/syslog
# درون less:
# /error را تایپ کنید تا "error" را جستجو کند
# n برای نتیجه بعدی
# G برای رفتن به انتها
# q برای خروج
```

## تمرین ۲: tail -f برای لاگ زنده
```bash
# در یک ترمینال:
tail -f /var/log/syslog

# در ترمینال دیگر:
logger "پیام آزمایشی"
# پیام را در ترمینال اول می‌بینید
```

## تمرین ۳: ترکیب head و tail
```bash
# خطوط ۵ تا ۱۰ یک فایل
head -n 10 /etc/passwd | tail -n 6

# آخرین ۵ خط /var/log/auth.log
tail -n 5 /var/log/auth.log
```

## تمرین ۴: ویرایش با nano
```bash
nano ~/notes.txt
# چند خط بنویسید
# Ctrl+O برای ذخیره
# Ctrl+X برای خروج
cat ~/notes.txt
```

### ✅ چک‌لیست یادگیری
- [ ] می‌توانم با less در فایل‌های بزرگ راحت کار کنم
- [ ] tail -f را برای لاگ زنده استفاده کرده‌ام
- [ ] با nano فایل ساخته و ذخیره کرده‌ام
