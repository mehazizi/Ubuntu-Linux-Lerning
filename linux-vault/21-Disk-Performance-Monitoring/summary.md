# فصل ۲۱ - پایش دیسک و عملکرد | خلاصه

## ابزارهای پایش
| ابزار | کاربرد |
|-------|---------|
| `df` | فضای دیسک |
| `du` | حجم پوشه‌ها |
| `free` | وضعیت حافظه |
| `vmstat` | آمار حافظه، CPU، I/O |
| `iostat` | آمار I/O دیسک |
| `sar` | ثبت و گزارش عملکرد |
| `uptime` | آپتایم و Load Average |
| `dmesg` | پیام‌های هسته |

## Load Average چیست؟
میانگین تعداد فرآیندهای در صف اجرا در ۱، ۵ و ۱۵ دقیقه گذشته:
```
load average: 1.20, 0.85, 0.90
              └1min  └5min  └15min
```
- اگر Load Average از تعداد CPU کمتر باشد: سیستم آرام
- اگر برابر باشد: سیستم در حال استفاده کامل
- اگر بیشتر باشد: سیستم تحت فشار

## انواع حافظه
| نوع | توضیح |
|-----|-------|
| total | کل حافظه فیزیکی |
| used | در حال استفاده |
| free | کاملاً خالی |
| buff/cache | استفاده‌شده برای کش (قابل آزادسازی) |
| available | قابل استفاده برای برنامه جدید |
