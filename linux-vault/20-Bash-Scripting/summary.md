# فصل ۲۰ - Bash Scripting | خلاصه

## ساختار اسکریپت
```bash
#!/bin/bash
# این خط اول (shebang) مفسر را مشخص می‌کند

set -euo pipefail  # تنظیمات امن
# -e: توقف در صورت خطا
# -u: خطا برای متغیر تعریف‌نشده
# -o pipefail: خطا در pipe

# کد اسکریپت
```

## انواع متغیر
```bash
NAME="مقدار"          # متغیر معمولی
readonly PI=3.14      # ثابت
LOCAL_VAR=$(command)  # خروجی دستور
export GLOBAL="val"   # متغیر محیطی
```

## آرگومان‌های اسکریپت
| متغیر | مقدار |
|-------|-------|
| `$0` | نام اسکریپت |
| `$1`, `$2` | آرگومان‌ها |
| `$#` | تعداد آرگومان‌ها |
| `$@` | همه آرگومان‌ها |
| `$?` | exit code دستور قبلی |
| `$$` | PID اسکریپت |

## Exit Code
- `0` = موفق
- `1-255` = خطا (عرف: `1` عمومی، `2` استفاده نادرست)
- `exit 0` یا `exit 1` در انتهای اسکریپت
