# فصل ۶ - مدیریت بسته | خلاصه

## مفاهیم اصلی
- **Repository**: مخزن آنلاین بسته‌های نرم‌افزاری
- **Package**: بسته نرم‌افزاری شامل فایل‌ها + متادیتا + اسکریپت نصب
- **Dependency**: وابستگی - بسته‌هایی که برای اجرا نیاز است
- **GPG Key**: کلید رمزنگاری برای تأیید صحت بسته‌ها

## لایه‌های مدیریت بسته در اوبونتو
```
apt (سطح بالا)     ← کار روزانه
    ↓
dpkg (سطح پایین)   ← نصب فایل .deb
    ↓
فایل‌سیستم         ← کپی فایل‌ها
```

## فایل‌های مهم
| فایل | کاربرد |
|------|---------|
| `/etc/apt/sources.list` | مخازن اصلی |
| `/etc/apt/sources.list.d/` | مخازن اضافه |
| `/var/cache/apt/archives/` | کش بسته‌های دانلودشده |
| `/var/lib/apt/lists/` | لیست بسته‌های مخازن |

## فرمت‌های نصب
| فرمت | روش نصب |
|------|---------|
| `.deb` | dpkg یا apt |
| `.tar.gz` | استخراج و make install |
| Binary | chmod +x و اجرا |
