# فصل ۴ - مشاهده و ویرایش فایل‌ها | خلاصه

## ابزارهای مشاهده فایل
| دستور | کاربرد | ویژگی خاص |
|-------|---------|------------|
| `cat` | نمایش کامل فایل | ساده و سریع |
| `less` | نمایش صفحه‌به‌صفحه | قابلیت جستجو با / |
| `more` | نمایش صفحه‌به‌صفحه | قدیمی‌تر از less |
| `head` | ابتدای فایل | پیش‌فرض ۱۰ خط |
| `tail` | انتهای فایل | -f برای دنبال‌کردن زنده |
| `nl` | نمایش با شماره خط | |
| `tac` | نمایش معکوس | از آخر به اول |
| `strings` | رشته‌های قابل خواندن از فایل باینری | |

## کلیدهای مهم less
| کلید | عملکرد |
|------|---------|
| `Space` یا `f` | صفحه بعد |
| `b` | صفحه قبل |
| `g` | ابتدای فایل |
| `G` | انتهای فایل |
| `/pattern` | جستجو |
| `n` | نتیجه بعدی |
| `q` | خروج |

## ویرایشگر nano
- ساده‌ترین ویرایشگر ترمینال
- کلیدها در پایین صفحه نشان داده می‌شوند
- `^` یعنی Ctrl
