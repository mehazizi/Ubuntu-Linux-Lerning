# فصل ۱۵ - لاگ‌ها | خلاصه

## سیستم‌های لاگ در لینوکس
| سیستم | توضیح |
|-------|-------|
| **journald** | لاگ دودویی systemd، سریع و قابل فیلتر |
| **rsyslog** | لاگ سنتی متنی، ارسال به سرور مرکزی |
| **logrotate** | چرخش و فشرده‌سازی لاگ‌ها |

## فایل‌های لاگ مهم
| فایل | محتوا |
|------|-------|
| `/var/log/syslog` | لاگ عمومی سیستم |
| `/var/log/auth.log` | احراز هویت و sudo |
| `/var/log/kern.log` | لاگ هسته |
| `/var/log/dpkg.log` | عملیات بسته‌ها |
| `/var/log/nginx/` | لاگ وب‌سرور |

## سطوح لاگ (از مهم‌تر به کم‌اهمیت‌تر)
```
0: emerg    - سیستم غیرقابل استفاده
1: alert    - اقدام فوری نیاز است
2: crit     - وضعیت بحرانی
3: err      - خطا
4: warning  - هشدار
5: notice   - رویداد عادی مهم
6: info     - اطلاعات
7: debug    - اطلاعات اشکال‌زدایی
```
