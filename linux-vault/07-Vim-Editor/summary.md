# فصل ۷ - ویرایشگر Vim | خلاصه

## چرا Vim؟
- در تمام سرورهای لینوکس موجود است
- بسیار سریع برای ویرایش متن
- بدون نیاز به رابط گرافیکی

## حالت‌های Vim
```
Normal Mode   ← حالت پیش‌فرض، حرکت و دستورات
    ↓  i, a, o
Insert Mode   ← تایپ متن
    ↓  Esc
Normal Mode
    ↓  :
Command Mode  ← ذخیره، خروج، جستجو
    ↓  v, V, Ctrl+V
Visual Mode   ← انتخاب متن
```

## دستورات حرکت (Normal Mode)
| کلید | حرکت |
|------|------|
| `h j k l` | چپ، پایین، بالا، راست |
| `w` | کلمه بعدی |
| `b` | کلمه قبلی |
| `0` | ابتدای خط |
| `$` | انتهای خط |
| `gg` | ابتدای فایل |
| `G` | انتهای فایل |
| `nG` | رفتن به خط n |
| `Ctrl+f` | صفحه بعد |
| `Ctrl+b` | صفحه قبل |

## دستورات ویرایش پرکاربرد
| کلید | عملکرد |
|------|---------|
| `dd` | حذف خط |
| `yy` | کپی خط |
| `p` | چسباندن |
| `u` | undo |
| `Ctrl+r` | redo |
| `x` | حذف کاراکتر |
| `r` | جایگزینی یک کاراکتر |
| `cw` | تغییر کلمه |
| `.` | تکرار آخرین دستور |
