# فصل ۵ - پردازش متن | خلاصه

## فلسفه لینوکس: ترکیب ابزارها
در لینوکس هر ابزار یک کار می‌کند اما با pipe (`|`) می‌توان آن‌ها را ترکیب کرد:
```
cat file | grep "error" | sort | uniq -c | sort -rn
```

## ابزارهای پردازش متن
| دستور | کاربرد |
|-------|---------|
| `grep` | جستجوی الگو در متن |
| `cut` | انتخاب ستون یا کاراکتر |
| `sort` | مرتب‌سازی |
| `uniq` | حذف تکراری‌ها |
| `wc` | شمارش خط، کلمه، کاراکتر |
| `tr` | جایگزینی کاراکتر |
| `paste` | ادغام ستونی فایل‌ها |
| `xargs` | تبدیل stdin به آرگومان |
| `tee` | نوشتن همزمان به فایل و stdout |
| `sed` | ویرایش متن با دستور |
| `awk` | پردازش ستونی متن |

## Pipe و Redirect
```
>    → خروجی به فایل (بازنویسی)
>>   → خروجی به فایل (افزودن)
<    → ورودی از فایل
|    → خروجی به دستور بعدی
2>   → خطا به فایل
&>   → هم خروجی هم خطا به فایل
```
