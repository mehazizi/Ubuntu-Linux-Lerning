# Chapter 01 — Lab Exercises

# هدف آزمایش‌ها

در پایان این آزمایش‌ها باید بتوانید:

- اطلاعات پایه سیستم را پیدا کنید.
    
- تفاوت Kernel، Distribution و Shell را در عمل مشاهده کنید.
    
- از سیستم Help لینوکس استفاده کنید.
    
- بدون مراجعه به اینترنت اطلاعات سیستم را استخراج کنید.

# Lab 01 — شناسایی سیستم‌عامل

هدف:

تشخیص Distribution و نسخه Linux.

دستورها:

```bash
cat /etc/os-release
```

```bash
lsb_release -a
```

سؤالات:

1. نام Distribution چیست؟
    
2. نسخه Ubuntu چیست؟
    
3. Codename چیست؟

# Lab 08 — Help System

هدف:

یادگیری پیدا کردن Help بدون اینترنت.

دستورها:

```bash
ls --help
```

```bash
man ls
```

```bash
man uname
```

```bash
apropos network
```

تمرین:

با استفاده از man پاسخ این سؤال‌ها را پیدا کنید:

1. دستور uname چه کاری انجام می‌دهد؟
    
2. گزینه `-a` چه کاربردی دارد؟
    
3. دستور date چه گزینه‌هایی دارد؟
